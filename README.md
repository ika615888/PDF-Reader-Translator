#  PDF翻訳アプリ

英語のPDFファイルを読み込み、自動で日本語に翻訳するデスクトップアプリです。  
PyQt5でGUIを構築し、MyMemory翻訳APIを使用して1ページずつ翻訳します。  
ClaudeやChatGPTなどのAIアシスタントの協力を得て個人開発しました。

---

##  主な機能

- PDFファイルをドラッグ＆ドロップで読み込み  
- 英語ページは自動的に日本語翻訳（日本語ページはそのまま表示）  
- ページごとの進捗表示バー  
- 翻訳結果をテキストファイルとして保存可能  
- MyMemory翻訳APIを使用（無料）  
- 翻訳エラー時の自動リトライ機能  
- 段落・文構造を自動整形して翻訳精度を向上  

---

##  必要なライブラリ

Python 3.8以上で動作確認済み。

```bash
pip install PyQt5 PyPDF2
```

---

##  使い方

1. 上記ライブラリをインストール  
2. このリポジトリをクローンまたはZIPダウンロードして展開  
3. ターミナルまたはコマンドプロンプトで、スクリプトを実行：

```bash
python main.py
```

4. アプリが起動したら、PDFファイルをウィンドウ中央のエリアにドラッグ＆ドロップ  
5. 英語ページが自動で翻訳され、日本語結果が下部に表示されます  
6. 翻訳結果は「保存」ボタンでテキストファイルとして保存可能です

##  仕様メモ

- 翻訳API： [MyMemory Translated API](https://mymemory.translated.net/doc/spec.php)  
- 英語判定：文字コード比率による簡易判定（英字70%以上で英語と判定）  
- 翻訳単位：文ごと（ピリオド・疑問符・感嘆符で分割）  
- 500文字以上は自動分割して送信  
- 各リクエスト間に0.3秒待機してレート制限を回避
